# BUG-009: tab-titles — precmd() écrase les precmd existants (oh-my-zsh, p10k)

**Type:** Bug
**Statut:** A faire
**Priorite:** Moyenne
**Complexite:** XS
**Tags:** cli
**Date creation:** 2026-03-21

---

## Description

L'installeur injecte une fonction `precmd()` dans `.zshrc` (ligne 179) qui écrase tout `precmd` existant d'autres outils (oh-my-zsh themes, powerlevel10k, etc.). Cela peut casser le prompt de l'utilisateur.

## Approche

Remplacer la définition directe par le hook zsh natif :
```bash
# AVANT
precmd() {
    printf "\033]1;%s\007" "$(basename "$PWD") — zsh"
}

# APRES
precmd_claude_conf() {
    printf "\033]1;%s\007" "$(basename "$PWD") — zsh"
}
autoload -Uz add-zsh-hook
add-zsh-hook precmd precmd_claude_conf
```

`add-zsh-hook` empile les fonctions au lieu de les écraser. Compatible oh-my-zsh, p10k, etc.
