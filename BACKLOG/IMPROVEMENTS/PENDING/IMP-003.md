**Type:** Improvement
**Statut:** A faire
**Priorite:** Basse
**Complexite:** XS
**Scope:** both
**Tags:** tooling, quality
**Date creation:** 2026-03-20

## Description

Le worker IMP-001 n'a pas pu lancer shellcheck car il n'etait pas installe. Pour un projet compose exclusivement de scripts shell, shellcheck est un outil de qualite fondamental qui devrait faire partie du setup dev.

## Taches

1. Ajouter une section "Development" ou "Contributing" dans le README.md root avec shellcheck comme pre-requis
2. Optionnel : ajouter un check shellcheck dans `critical-thinking/install.sh` step 1 (dependencies) — en tant que warning, pas en bloquant (shellcheck est un outil dev, pas un runtime dep)

## Criteres d'acceptation

- [ ] Le README mentionne shellcheck comme outil dev recommande
- [ ] La commande d'installation est documentee (`brew install shellcheck` / `apt install shellcheck`)

## Fichiers concernes

- `README.md` (root)
