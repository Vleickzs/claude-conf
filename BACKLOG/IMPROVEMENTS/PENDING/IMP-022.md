## Description

Integrer le hook `auto-start.sh` dans le module `setup-project/`. Quand `/setup-project` s'execute, il installe un hook UserPromptSubmit qui charge automatiquement le contexte au premier message de chaque session (git status, backlog, mention handoff). Ca remplace le besoin de taper `/start` manuellement.

`/start` reste disponible pour recharger le contexte en cours de session.

Le hook est base sur `~/.claude/hooks/auto-start.sh` existant.

## Fichiers concernes

- `setup-project/hooks/auto-start.sh` (creer)
- `setup-project/install.sh` (modifier — ajouter install du hook + config settings.json)
