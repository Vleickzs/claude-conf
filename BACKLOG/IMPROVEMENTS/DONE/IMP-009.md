# IMP-009: install.sh — modules invisibles dans MODULES array + README

**Type:** Improvement
**Statut:** A faire
**Priorite:** Moyenne
**Complexite:** XS
**Tags:** cli, ux
**Date creation:** 2026-03-21

---

## Description

Le MODULES array dans `install.sh` et la table du README ne listent pas tous les modules existants. pre-commit-gate, critical-thinking et d'autres modules sont installables mais invisibles pour un nouvel utilisateur qui suit le README ou le menu interactif.

## Approche

1. Lister tous les répertoires contenant un `install.sh` (= modules réels)
2. Mettre à jour le MODULES array, DESCRIPTIONS et DEPS dans `install.sh`
3. Mettre à jour la table des modules dans README.md (FR + EN)
4. Vérifier la cohérence entre les deux

Note : si IMP-007 (suppression oneshot) est fait avant, ne pas inclure oneshot.

## Fichiers concernes

- `install.sh` — arrays MODULES, DESCRIPTIONS, DEPS
- `README.md` — table des modules

## Depends on

- IMP-007 (supprimer oneshot) — pour savoir quels modules lister

## Criteres d'acceptation

- [ ] Tous les modules existants apparaissent dans install.sh ET README
- [ ] Le menu interactif propose tous les modules
- [ ] Les descriptions sont cohérentes entre install.sh et README

## Tests de validation
